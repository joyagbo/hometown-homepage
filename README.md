# Kainji Tourism Portal

## Project Overview

This project is a **solo assignment** for my web development course. It is a landing page dedicated to **Kainji**, a unique town in **Niger State, Nigeria**. The website highlights the town's **history, wildlife, and engineering significance**, with a strong focus on:

* **Kainji Lake National Park**
* **Kainji Hydroelectric Dam**

The primary goal of this project was to practice **HTML5 semantic structure** and **CSS3 Flexbox** to create a **responsive, modern layout**.

---

## Features

* **Hero Section**
  A high-impact introduction featuring a badge-style title and a highlighted subtext box.

* **Activities Grid**
  A three-column Flexbox layout showcasing top tourist attractions:

  * Wildlife safari at Kainji Lake National Park
  * Educational tours of the Kainji Hydroelectric Dam
  * Cultural exploration of the Borgu Emirâ€™s Palace

* **Interactive Design**
  Image hover effects to improve user engagement and visual appeal.

* **Guide Section**
  A specialized card layout featuring a local tour guide, built using nested Flexbox containers.

* **Semantic Footer**
  A clean, well-structured footer containing copyright information and social links.

---

## Technologies Used

* **HTML5**
  Used for page structure with semantic elements such as `<header>`, `<main>`, `<section>`, and `<footer>`.

* **CSS3**
  Used for styling and layout, including:

  * **Flexbox** for alignment and responsiveness
  * **Custom color palette** featuring *Deep River Blue* (`#003B5C`) and *Savannah Gold* accents
  * **Hover effects** for interactivity

* **Google Fonts**

  * *Lato* font family, imported via a link in the `<head>` section
